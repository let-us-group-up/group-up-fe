## Installing

Before using this application you have to install packages

    yarn install


## Start dev server

    yarn start


## Build project

    yarn build


## Lint files

    yarn lint


## Run tests

### Unit tests

Current there are no unit tests

    yarn test


## Create tag

    yarn version
    yarn pushTag

    0.0.1 - patch version
    0.1.0 - minor version
    1.0.0 - major version


## VS code settings

With this [VS code settings](https://code.visualstudio.com/docs/getstarted/settings), the code editor can format the code on save

    {
        "javascript.format.enable": false,
        "editor.codeActionsOnSave": {
            "source.fixAll": true
        }
    }
